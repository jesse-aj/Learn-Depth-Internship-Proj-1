# Quiz Game Application (CLI)

##  Project Description
This is a simple Command Line Interface (CLI) Quiz Game built using Python.  
The application asks multiple-choice questions, evaluates user answers, and displays the final score.

The quiz runs entirely in the terminal.

---

##  Features Implemented

- 5 multiple-choice questions
- Accept user answers from keyboard
- Check correct and incorrect answers
- Maintain score during quiz
- Display final result
- Option to restart or exit

##  How It Works

- Questions are stored in a list of dictionaries.
- Each dictionary contains:
- Question
- Options
- Correct answer
- A loop is used to go through each question.
- The score variable increases for correct answers.
- A menu allows the user to restart or exit.
